Kemuri Craft

Let the Chaos begin!

About

Kemuri Craft is an Anarchy Minecraft server, with as much freedom as possible. There are no world vipes, the gameplay is pure vanilla, and there are no pesky anti-cheats.

How to play?

Kemuri Craft is hosted on my laptop without a public address. In order to connect to Kemuri Craft:
  1. Use Minecraft Java, version 1.21.5, no need for a Minecraft license
  2. Install the following mod (e.g. from Modrinth): Modflared
  3. Create a new server with the address "play.kemuri.life"
  4. Join and enjoy the Chaos!

Rules

Since the server is a pure Anarchy you are allowed to do basically anything. There are some things which are not welcome, though:
  • Spawn-killing players
  • Building lag machines or using modified clients to lag the server
  • Abusing vulnerabilities of the login system
  • Using inappropriate username or skin
The punishment will be based on the severity of the problem, and can be executed by the server admin "gXLg" without any further explanations.
